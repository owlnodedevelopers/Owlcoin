Sample configuration files for:
```
SystemD: owlcoind.service
Upstart: owlcoind.conf
OpenRC:  owlcoind.openrc
         owlcoind.openrcconf
CentOS:  owlcoind.init
macOS:    org.owlcoin.owlcoind.plist
```
